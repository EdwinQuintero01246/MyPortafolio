  • Veja Delivery

    Trabajos que realice full backend, frontend y analista de la base de datos.

    Esta aplicación se creó para la clase de sistemas expertos, es una app delivery el cual esta diseñada para brindar este tipo de servicios, es responsive está pensado para dispositivo de resolución pequeña teléfonos u ordenadores, consta de tres tipos de usuarios administrador, motorista, cliente.
    El administrador es le encargado de crear categorías, ingresar empresas, productos, precios, sucursales, aprobar los usuarios para que sean motorista, etc.
    El motorista puede gestionar pedidos ir cambiando el estado del mismo, completar entregas, etc.
    El cliente este puede cambiar su información, agregar formas de pago de tarjeta este mismo, simulamos compras, ordenar, agregar carritos, generar compras de todos los objetos que están en carrito de compras, obtener latitud y longitud con un BoxMap, etc.
